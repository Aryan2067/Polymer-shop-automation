# Polymer Shop Automation

Selenium automation suite built using:

- Java
- Selenium WebDriver 4
- TestNG
- Edge Browser
- Selenium Manager (No driver setup required)

## Test Coverage

- Home page validation
- Navigation (Men / Women)
- Product listing
- Add to cart
- Cart validation
- Quantity update
- Remove item
- Browser navigation
- Footer validation

## How to Run

1. Add Selenium + TestNG jars
2. Open in VS Code
3. Run using TestNG
